## ğŸ› ï¸ Tech Stack

This project was built from scratch without any UI libraries like Bootstrap or Tailwind.

* **Frontend:** React.js + Vite
* **AI Model:** NVIDIA Llama 3.1 8B Instruct (via NVIDIA NIM)
* **OCR Engine:** Tesseract.js (Images) + PDF.js (Documents)
* **Styling:** Pure CSS3 (Variables, Keyframes, Glassmorphism)
* **Animations:** Framer Motion + Lucide React Icons
* **Notifications:** React Hot Toast
